**New Holland T6.180** — це топова модель серії T6 — найпопулярнішої серії тракторів New Holland уже тривалий час.

## Двигун FPT NEF 6,7 л

T6.180 приводить шестициліндровий **FPT NEF 6,7 л** потужністю 180 коней (132 кВт), що відповідає нормі викидів **Stage V**. Шестициліндровий двигун у цьому універсальному класі працює спокійніше за чотирициліндровий і пропонує плавніший перебіг крутного моменту — для оператора це означає тихіший хід і кращу тягу на низьких обертах.

## Трансмісія — Electro Command або Auto Command

Клієнт обирає між двома трансмісіями. **Electro Command** — це напівавтоматична 16/16 із перемиканням під навантаженням — перевірена й доступна за ціною. **Auto Command** — це безступінчаста (CVT) трансмісія з плавною зміною швидкості без ступенів перемикання, що підходить для змінних умов і частих маніпуляцій із фронтальним навантажувачем.

## Універсальне використання

T6.180 розроблено як всебічний трактор — він впорається з підготовкою ґрунту, сівбою, роботою із збиральним возом і прес-підбирачем, транспортуванням та обслуговуванням фронтального навантажувача. Саме ця універсальність робить серію T6 найпопулярнішою серією New Holland уже тривалий час.

## Для кого призначений T6.180

T6.180 призначений для середніх сімейних і більших ферм, яким потрібен один надійний універсал для більшості робіт. Для ферм із власним виробництвом біогазу New Holland пропонує в серії T6 також модель на біометані — T6 Methane Power.
